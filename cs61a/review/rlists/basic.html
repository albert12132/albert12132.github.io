We will be using the following implementation of immutable recursive lists. Keep in mind that your code should not depend on the assumption that rlists are implemented as tuples -- preserve data abstraction! 
empty_rlist = None

def rlist(first, rest=empty_rlist):
    return (first, rest)

def first(s):
    return s[0]

def rest(s):
    return s[1]

Concept Questions

Question 1:

What type of object can first be (e.g. int, string, function, etc.)? What type of object can rest be?

first can be any type of object, even an rlist. rest can only be an rlist.

Question 2:

Which of the following are valid Rlist constructors?

  1. rlist(1, 3)
  2. rlist(1, None)
  3. rlist(1, rlist(4, empty_rlist))
  4. rlist(1, empty_rlist)
  5. rlist()

The correct answers are in bold:

  1. rlist(1, 3)
  2. rlist(1, None)
  3. rlist(1, rlist(4, empty_rlist))
  4. rlist(1, empty_rlist)
  5. rlist()

Code-Writing Questions

Question 7:

Implement a function rlist_to_tup that takes an rlist as an argument, and returns a tuple that contains the same elements as the rlist.

def rlist_to_tup(lst):
    """Returns a tuple that contains the same elements as the
    rlist.

    >>> r = rlist(1, rlist(2, rlist(3, empty_rlist)))
    >>> rlist_to_tup(r)
    (1, 2, 3)
    """
    "*** YOUR CODE HERE ***"

A recursive version:

def rlist_to_tup(lst):
    if lst == empty_rlist:
        return ()
    return (first(lst),) + rlist_to_tup(rest(lst))

An iterative version:

def rlist_to_tup(lst):
    new = ()
    while lst != empty_rlist:
        new += (first(lst),)
        lst = rest(lst)
    return new

Question 8:

Implement a function map_rlist that maps a function f onto each element of an rlist.

def map_rlist(lst, f):
    """Maps f onto each element in the rlist.

    >>> r = rlist(1, rlist(2, rlist(3, empty_rlist)))
    >>> rlist_to_tup(map_rlist(r, lambda x: x**2))
    (1, 4, 9)
    """
    "*** YOUR CODE HERE ***"

A recursive version:

def map_rlist(lst, f):
    if lst == empty_rlist:
        return empty_rlist
    return rlist(f(first(lst)), map_rlist(rest(lst), f))

An iterative version:

def map_rlist(lst, f):
    new = empty_rlist
    while lst != empty_rlist:
        new = rlist(f(first(lst)), new)
        lst = rest(lst)
    while new != empty_rlist:
        lst = rlist(first(new), lst)
        new = rest(new)
    return lst
