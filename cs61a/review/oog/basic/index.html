Conceptual

Q1

What is the time complexity of this function in big-Theta (θ) notation?

def one(n):
    for a in range(n):
        for b in range(n/2):
            for c in range(n/4):
                print(a + b + c)

Answer: θ(n3)

Q2

What is the time complexity of this function in big-Theta (θ) notation?

def two(n):
    for a in range(n):
        for b in range(1000000000):
            for c in range(n):
                print(a + b + c)

Answer: θ(n2)

Q3

What is the time complexity of this function in big-Theta (θ) notation?

def three(n):
    while n > 1:
        result = n * n
        print(result)
        n = n / 10
    return False

Answer: θ(logn)

Q4

What is the time complexity of this function in big-Theta (θ) notation?

def four(lst):
    if len(lst) < 12345:
        return lst[0]
    return four(lst[1:])

Answer: θ(n), where n is the length of the list.

Q5

What is the time complexity of this function in big-Theta (θ) notation?

def five(n):
    def helper(x):
        return x + n
    return helper(n/2)

Answer: θ(1)

Q6

What is the time complexity of this function in big-Theta (θ) notation?

def reverse(lst):
    if not lst:
        return []
    result = reverse(lst[1:])
    result.append(lst[0])
    return result

Answer: θ(n), where n is the size of the list.
