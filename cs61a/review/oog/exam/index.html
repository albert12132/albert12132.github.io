Conceptual

Q1

Find the time complexity of main in big-Theta (θ) notation.

def helper(x):
    for i in range(x):
        print(i)

def main(n):
    if n == 2:
        return 0
    else:
        return helper(n - 1) + helper(n - 2)

Answer: θ(n)

Q2

Find the time complexity of bar in big-Theta (θ) notation.

def foo(x):
    for i in range(x):
        for j in range(x):
            print(x)

def bar(n):
    while n > 0:
        foo(100000)
        n -= 1

Answer: θ(n)

Q3

Find the time complexity of funny in big-Theta (θ) notation.

def joke(n):
    for i in range(n**2):
        print(i)

def funny(n):
    for i in range(n**2):
        print(joke(100))
    return 'haha'

Answer: θ(n2)
