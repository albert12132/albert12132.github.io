Question 1:

What is the time complexity of this function in big-Theta (&theta;) notation?

def one(n):
    for a in range(n):
        for b in range(n/2):
            for c in range(n/4):
                print(a + b + c)

Question 6:

What is the time complexity of this function in big-Theta (&theta;) notation?

def reverse(lst):
    if not lst:
        return []
    result = reverse(lst[1:])
    result.append(lst[0])
    return result
