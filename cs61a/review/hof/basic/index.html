Conceptual

Q1

What is the definition of a higher-order function?

Answer: A function is a higher-order function if it satisfies at least one of the following:

  • It takes at least one function as an argument
  • It returns a function

    • Q2

    In the following code, what type of object (number, boolean, string, function) does foo return? What type of object should x be for this function to work?

    def foo(x):
    def inner(y):
        return x(y)
    return inner

    Answer: foo should return a function object (in particular, the function inner). x should also be a function, or else calling inner will result in an error (because it will try x(y)).
