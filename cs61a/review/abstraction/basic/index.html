Conceptual

Q1

What is a constructor? What is a selector?

Answer: A constructor is a function that creates an abstract object. A selector is a function that retrieves information from a specified object.

Q2

The following is the rlist abstract data type from lecture. Which functions are constructors? Which functions are selectors?

empty_rlist = ()

def rlist(first, rest):
    return (first, rest)

def first(r):
    return r[0]

def rest(r):
    return r[1]

Answer: There is one constructor: rlist. There are two selectors: first and rest.

Code Writing

Q1

Implement a function midpoint that calculates the midpoint of two points p1 and p2. These two "point objects" are represented by an abstract data type, shown below. works:

def point(x, y):
    return (x, y)

def x_coord(p):
    return p[0]

def y_coord(p):
    return p[1]

def midpoint(p1, p2):
    """Doctests:

    >>> p1, p2 = point(1, 1), point(3, 3)
    >>> p3 = midpoint(p1, p2)
    >>> x_coord(p3)
    2
    >>> y_coord(p3)
    2
    """
    "*** YOUR CODE HERE ***" 
def midpoint(p1, p2):
    mid_x = (x_coord(p1) + x_coord(p2)) / 2
    mid_y = (y_coord(p1) + y_coord(p2)) / 2
    return point(mid_x, mid_y)
